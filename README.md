# Weather Analytical Dashboard

A modern and responsive weather analytics dashboard built using HTML, CSS, and JavaScript.  
The application fetches real-time weather data and displays temperature trends, humidity, wind speed, and a 7-day forecast in a clean dashboard UI.

## Features
- City-based weather search
- Location-based weather using browser geolocation (with user permission)
- Real-time weather data integration
- 7-day weather forecast
- Interactive temperature chart using Chart.js
- Loading states and error handling
- Responsive dashboard UI

## Tech Stack
- HTML
- CSS
- JavaScript (ES6)
- Chart.js
- REST Weather API

## Setup Instructions
This project uses a third-party Weather API.
